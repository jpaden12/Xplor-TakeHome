# Xplor Take Home Coding Challenge 

## How to run: 
* Clone this repository. This will include the lib and src folders. 
* After cloning, you will get a folder named Xplor-TakeHome. 

### IntelliJ
* With IntelliJ, open the Xplor-TakeHome folder. Do not open the src folder inside of it.
* JUnit 4 is included in the repository.
* You will be able to run the tests in src/test/PersonTest.java without using Maven or another build tool. Use the green play button
at the top of the IDE to run the tests. 

### Visual Studio Code 
* With Visual Studio Code, open the Xplor-TakeHome folder. Do not open the srrc folder inside of it.
* Install the following VS Code extensions.
  - Project Manager for Java
  - Extension pack for Java
  - Debugger for Java
  - Test runner for Java
* The tests in src/test/PersonTest.java should be available for you to run. Use the green checkmark in the editor
  or the play button in the testing menu to run the tests.
